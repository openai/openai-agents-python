# Support Transcript Summary

## Overview

Use this skill to condense a support conversation into a short summary and actionable next steps.

## Checklist

1. Capture the customer issue and desired outcome.
2. Note key troubleshooting steps already taken.
3. List pending actions and required follow-up details.
