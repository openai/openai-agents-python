# Repo Architecture Mapper

## 任務
- 掃描專案結構，列出：入口點、主要模組/層次、測試位置、建置/部署設定
- 與同類專案的常見模式比對，提出命名/分層改進建議（純文字）

## 輸出
- 簡潔清單（含路徑）＋ 1–2 段建議

## 使用情境
- 「請分析這個專案的整體架構與關鍵模組」
- 「這次變更會影響哪些元件？」
